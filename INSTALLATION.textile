h1. Installation

Installation is pretty straight-forward.
The ".grimreporc" configuration file should reside in the user home director. 
And the repo_* programs should be placed in a directory listed in $PATH

h3. Recommendations

To make sure that ssh commands works without user involement,
please make sure public key authentication is activated and working.
